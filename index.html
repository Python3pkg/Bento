Bento documentation (release 0.0.3dev8590e92813)

Welcome to bento main documentation. Bento is a no-nonsense packaging tool solution for python softwares. Packaging simple python code consists in writing a simple declarative text file. Bento aims at being simpler, more pythonic and more extensible than existing python packaging tools.
